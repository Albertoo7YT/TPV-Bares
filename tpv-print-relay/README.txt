1. Ejecute install-service.bat como administrador
2. Abra el navegador en http://localhost:9191
3. Configure la URL del servidor y las impresoras
4. El programa arrancara automaticamente con Windows

Notas:
- No necesita Node.js instalado en el PC destino
- El archivo config.json se crea junto al ejecutable instalado
- Si el icono de bandeja no aparece, el panel local y las notificaciones siguen funcionando
